## 🛠️ Sysmon

### *Standard Logging for Cross-Platform Security*

> ### 🔍 Why Sysmon
>
> Sysmon is used as the **standard logging system** for PLURA.
> It delivers **consistent and high-fidelity security logs** across **Windows and Linux**,
> enabling **unified detection, investigation, and response**.
>
> To maintain a **common audit baseline** and **cross-platform visibility**,
> using Sysmon on **both operating systems is strongly recommended**.

---

> ### 🔍 Sysmon을 사용하는 이유
>
> Sysmon은 PLURA의 **표준 로깅 시스템(Standard Logging System)** 입니다.
> Windows와 Linux 환경 전반에 걸쳐
> **동일한 보안 이벤트 기준과 일관된 로그 구조**를 제공합니다.
>
> 이를 통해 **탐지 · 분석 · 대응을 하나의 기준으로 통합**할 수 있으며,
> **운영 환경에서는 Windows와 Linux 모두에 Sysmon 사용을 권장**합니다.
